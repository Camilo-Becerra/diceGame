Interactive Dice Betting Game
A browser-based dice betting game built with vanilla JavaScript, HTML5, and SVG graphics. Players bet money on the outcome of two dice rolls, with animated dice and real-time win/loss feedback.
Show Image
How to Play

Select a bet amount using the buttons
Click a number (2–12) to bet on the outcome of the two dice
Watch the dice animate and roll
Win or lose money based on whether your number matches the total rolled
You start with $100 — don't go broke!

Features

Animated SVG dice that roll from randomized starting positions
Dynamically generated dots using 2D arrays to represent each die face
Real-time cash tracker that updates after every roll
Win/loss message with a 500ms delay timed to appear after the dice settle
Hover effects on betting numbers for better interactivity
Input validation — can't roll without placing a bet, can't bet more than you have
